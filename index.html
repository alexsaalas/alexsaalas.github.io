Proyecto destacado

Recetas360

Mejor proyecto del curso de CFGS DAM

Aplicación online donde los usuarios pueden subir recetas de cualquier tipo de gastronomía y compartirlas con otros usuarios. Cada receta puede ser consultada, valorada, compartida y reseñada, generando una experiencia colaborativa en torno a la cocina.

Además, la aplicación incorpora funcionalidades avanzadas como consulta paso a paso de la receta, cronómetro de cocción con alarma automática en el móvil, sistema de valoración por estrellas, tabla de calorías mediante integración con API y un carrito de la compra que consulta precios de supermercados para calcular el coste total de los ingredientes.

A nivel técnico, la información se almacena en Google Firebase, permitiendo persistencia online de recetas y acceso por parte de distintos usuarios. También integra la API de Open Food Facts para la consulta nutricional.

IA aplicada

PhisingIA

Detector de phishing de correos

Aplicación orientada a detectar intentos de phishing mediante inteligencia artificial. El usuario puede analizar contenido de dos formas: escribiendo texto directamente o subiendo una imagen.

A partir de esa entrada, el sistema procesa la información y determina si el contenido analizado corresponde a un posible intento de phishing o no, devolviendo un resultado claro y directo: “Es phishing” o “No es phishing”.

Este proyecto ha sido desarrollado en Python utilizando Gemini como motor de IA, reflejando mi interés por combinar desarrollo de aplicaciones con casos de uso reales relacionados con análisis de contenido y seguridad.

Machine Learning

FraudShield

Sistema de detección de fraude en transacciones

Proyecto de machine learning desarrollado en Python para detectar transacciones potencialmente fraudulentas en un dataset altamente desbalanceado.

El sistema incluye carga y análisis exploratorio de datos, preprocesamiento, feature engineering, entrenamiento y comparación de modelos, además de evaluación avanzada con métricas como ROC-AUC, PR-AUC, precision y recall.

Se compararon modelos como Logistic Regression y Random Forest, logrando con este último una mejora clara en la capacidad de detección de fraude. También se trabajó el ajuste del threshold de decisión, validación cruzada y análisis de importancia de variables para acercar el proyecto a un enfoque más profesional.
